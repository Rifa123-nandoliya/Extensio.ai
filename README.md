# 🚀 Extensio AI

A No-Code Chrome Extension Generator powered by AI.

## ✨ Features
- Generate Chrome extensions using natural language
- Modern UI with sidebar navigation
- Downloads history using localStorage
- Contact form for user feedback

## 🛠️ Tech Stack
- React (Frontend)
- Node.js (Backend)
- Vite
